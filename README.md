# Pareto: Quantity Breaks & Discounts — CS Training

Internal training materials for the Customer Support team on the **Pareto: Quantity Breaks & Discounts** Shopify app.

## Training Structure

| Day | File | Topics | Slides |
|-----|------|--------|--------|
| **Day 1** | `pareto_training_day1_v4.html` | Quantity Breaks, One Discount, General Settings, Troubleshooting | 38 |
| **Day 2** | `pareto_training_day2.html` | CS Essentials, Order Discount, Buy X Get Y | 38 |
| **Day 3** | `pareto_training_day3.html` | Bundles (FBT, Traditional, Single), Checkout Upsell, Boost Conversion | 36 |
| **Day 4** | `pareto_training_day4.html` | App Configurations, Advanced Topics, Troubleshooting Mastery, Quick Reference | 35 |

**Total: 147 interactive slides** covering the full Pareto app.

## Features

- Interactive exercises: quizzes, true/false, matching, bingo, spinwheel, ticket simulations, decision trees
- Broken-setup diagnosis exercises for every offer type
- Bilingual support (Vietnamese / English) with toggle button (Days 2-4)
- Cross-day navigation bar linking all training files
- Image placeholders ready to be filled with app screenshots
- Fully self-contained HTML files — no server or build step required

## Day-by-Day Content

### Day 1 — Foundations
- How Pareto discounts differ from native Shopify discounts
- Three offer groups: Spent-based, Product-based, Quantity-based
- General Settings: campaign naming, discounted products, quantity conditions, target customers, schedule, markets
- Quantity Breaks: 7 templates, 5 layouts, multiples, advanced settings
- One Discount: main rule + 4 sub-rules, multiples, launch time
- Widget troubleshooting checklist

### Day 2 — CS Workflow + Core Offer Types
- CS vs Dev responsibilities and escalation process
- Top 10 merchant questions with quick answers
- Discount code + Pareto interaction (stacking rules)
- Priority rules when offers overlap
- Order Discount: value-based triggers, customer buy conditions, free gift/shipping, multiples + max cap
- Buy X Get Y: cross-sell, free gifts (same/different), discount source

### Day 3 — Bundles + Conversion Tools
- Frequently Bought Together, Product Bundle, Single Bundle
- Checkout Upsell: checkout/thank-you/order-status pages, suggest logic
- Boost Conversion: discounted price display, banner, popup, collection page, auto translate

### Day 4 — Mastery
- App Configurations: Automatic vs Manual discount method, Product Data Sync, Custom CSS/Classname
- Advanced deep dives: priority rules, launch time countdown behaviors, maximum discount limits, quantity counting matrix
- Interactive troubleshooting decision trees
- Discount stacking conflicts, theme compatibility, subscription app conflicts
- Master cheat sheet and escalation matrix

## Usage

Open any `.html` file in a modern browser. Use arrow keys or on-screen navigation to move between slides. Use the Day navigation bar at the top to switch between training days.
