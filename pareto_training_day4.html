Module 13 · ⚠️ Auto Translate

Auto Translate — rule quan trọngAuto Translate — critical rule

🚨
Auto Translate KHÔNG dịch ngôn ngữ DEFAULT của store. Nó chỉ dịch các ngôn ngữ non-default được khách switch sang ở storefront.Auto Translate does NOT translate the store's DEFAULT language. It only translates non-default languages that customers switch to on storefront.

Vì sao?Why?

Default language được coi là "nguồn" — text merchant nhập vào widget settings mặc định là default language. Auto Translate dịch từ default → các ngôn ngữ khác. Không có "nguồn khác" để dịch sang default.The default language is the "source" — text merchant enters in widget settings is assumed to be the default language. Auto Translate translates FROM default → other languages. There's no other source to translate TO the default.

Cách fix khi default language text saiHow to fix default language text

Option A
Merchant đổi trực tiếp text trong widget settings (Custom text trong Display Settings). Đây là nguồn → sẽ hiện cho default language.Merchant directly edits text in widget settings (Custom text in Display Settings). This IS the source → shows on default language.
Option B
Dùng Shopify Translate & Adapt để override text cho default language. App tôn trọng Shopify translations.Use Shopify Translate & Adapt to override text for default language. App respects Shopify translations.
⚠️
Ticket phổ biến: "Dutch dịch OK, English không dịch đúng". Case thực tế: store default = English → app không auto-translate English. Hướng dẫn merchant dùng Option A hoặc B ở trên.Common ticket: "Dutch translates fine, English doesn't translate correctly". Real case: store default = English → app doesn't auto-translate English. Guide merchant to Option A or B above.
